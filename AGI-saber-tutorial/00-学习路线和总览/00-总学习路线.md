# 00-总学习路线

## 1. 一句话结论

学习 AGI-saber 不要从细节开始。

推荐顺序：

```text
先主链路
再记忆系统
再工具调用
最后面试速背
```

---

## 2. 第一阶段：主链路

目标：

```text
知道一个 query 从哪里进来，经过哪些核心对象，最后怎么变成 answer。
```

要能回答：

```text
Controller 是入口吗？
UnifiedAgentService 做什么？
memPrefix 是什么？
histMsgs 是什么？
怎么决定 chat/tool/react/rag？
回答后做了什么？
```

---

## 3. 第二阶段：记忆系统

目标：

```text
知道短期、偏好、长期、图记忆分别解决什么问题。
```

要能回答：

```text
召回是不是异步？
长期记忆什么时候写？
score 怎么算？
图记忆怎么召回？
consolidation 为什么需要？
内存、DB、Neo4j 怎么最终一致？
```
